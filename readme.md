### 洛谷冬日绘版脚本

##### 使用方法

在`luogu.py`中更改配置

`width，height`:图片的宽和高

`x1,y1`:图片起始点的偏移量

`cookies`:把你拥有的所有cookies填入

`stoptime`:建议30/cookies数量

`homepageurl`:https://luogu.com.cn

`imagepath`:把你的图片拷贝到根目录，然后更改为你的图片名

然后运行即可
